DynaMiTe: Towards robust 2D Gaussian Splatting based camera localization

Abstract

In this work we present DynaMiTE, a robust and probabilistic framework for RGB-D SLAM based on 2D Gaussian Splatting. Building upon recent advances in novel view synthesis and differentiable rendering, our method introduces a learned Gaussian Process module to continuously disentangle dynamic from static regions, improving localization robustness in challenging environments.

Dynamics Masking Module

To handle dynamic scenes, we introduce a probabilistic Gaussian Process module. For each new keyframe, we compute a robust residual between the original and rendered images:

\begin{equation} r = (1 - C)(1 - L)(1 - S), \end{equation}

where \(C, L, S\) denote the contrast, luminance, and structure components of the SSIM loss. This residual captures subtle variations that reveal dynamic regions.

We assume the residual is parameterized by semantic features \(z \in \mathbb{R}^d\) extracted using DINOv2 embeddings. The dataset \(\mathcal{D} = \{(z_i, r(z_i))\}\) is used to train a Gaussian Process with a Matérn + RBF kernel. A variational approach minimizes the evidence lower bound (ELBO) via stochastic gradient descent. To keep the model compact, we perform active point selection, retaining only those with the highest uncertainty within a sliding window. The final binary mask is obtained by thresholding the exponential moving average of the predicted mean.
